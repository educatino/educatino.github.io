Curso Intensivo: Armado y Reparación de PC

Descripción:

El curso esta dirigido a todos aquellos que deseen para si mismo o como salida laboral, reparar PCs, armarse una nueva o actualizar una vieja. Tiene como particularidad ser intensivo y esencialmente practico. Durante las dos semanas que dura el curso se aprenderá a diagnosticar y reparar tanto problemas de hardware como de software. Ademas, también serán capaces de seleccionar el hardware apropiado de modo de configurar y ensamblar computadoras destinadas para su venta.

Metodología:

En cada clase se dispondrá de tres horas, dos de las cuales se las emplearan para reparar y desarrollar ejemplos prácticos (pueden traer comutadoras para reparar)

(Vacaciones de invierno)
